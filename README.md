# Scripting Project – Data Science Analyse

Dieses Projekt wurde im Rahmen des MAS Data Science erstellt und beinhaltet verschiedene Analysen mit Python und Jupyter Notebooks.

Der Fokus liegt auf der Aufbereitung, Analyse und Visualisierung von Daten, insbesondere im Kontext von Mobilität und geografischen Informationen.

---

## 📊 Projektinhalt

Das Repository enthält mehrere Jupyter Notebooks, welche unterschiedliche Aspekte der Datenanalyse behandeln:

* **01_Data_Preparation.ipynb**
  Datenaufbereitung und Bereinigung der Rohdaten

* **02_Plots_Fahrgastzahlen.ipynb**
  Visualisierung von Fahrgastzahlen mittels Diagrammen

* **03_Basiskarten für Detailanalysen.ipynb**
  Erstellung von Karten als Grundlage für geografische Analysen

* **04_Dicht besiedelte Adressen.ipynb**
  Analyse von Adressdaten zur Identifikation dicht besiedelter Gebiete

Zusätzlich enthält der Ordner `Data/` die verwendeten Datensätze.

---

## 🎯 Ziel des Projekts

Ziel ist es, praxisnah zu zeigen, wie:

* Daten vorbereitet und bereinigt werden
* explorative Datenanalyse durchgeführt wird
* Visualisierungen erstellt werden
* geografische Daten analysiert und interpretiert werden

---

## ⚙️ Voraussetzungen

Für die Ausführung werden folgende Tools benötigt:

* Python 3.9 oder neuer
* Jupyter Notebook oder JupyterLab

Empfohlene Python Libraries:

* pandas
* numpy
* matplotlib
* seaborn
* geopandas
* folium (falls Karten verwendet werden)

---

## 🚀 Installation

Repository klonen:

```bash
git clone https://github.com/niciamcompi/scripting-project.git
cd scripting-project
```

Virtuelle Umgebung erstellen (optional, empfohlen):

```bash
python -m venv venv
source venv/bin/activate   # Mac/Linux
venv\Scripts\activate      # Windows
```

Abhängigkeiten installieren:

```bash
pip install -r requirements.txt
```

Falls keine requirements.txt vorhanden ist, können die benötigten Pakete manuell installiert werden.

---

## ▶️ Verwendung

Jupyter Notebook starten:

```bash
jupyter notebook
```

Dann eines der Notebooks öffnen, z. B.:

* `01_Data_Preparation.ipynb`
* `02_Plots_Fahrgastzahlen.ipynb`
